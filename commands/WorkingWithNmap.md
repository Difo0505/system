# Fast scan

```
nmap -sS <ip>
```

# Scan port

```
nmap -p <port> <ip>
```

# Scan tcp

```
nmap -sT <ip>
```

# Scan udp

```
nmap -sU <ip>
```
It takes a long time to complete the task.

# Scan service version

```
nmap -sV <ip>
```

# Scan supported protocol

```
nmap -sO <ip>
```
It takes a long time to complete the task.

# Scan OS fingerprint

```
nmap -sV -O <ip>
```

# Scan network

```
nmap -sS 192.168.10.0/24
```
